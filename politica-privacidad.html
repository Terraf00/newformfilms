Política de privacidad

1. Responsable del tratamiento

En cumplimiento de lo dispuesto en el Reglamento (UE) 2016/679 del Parlamento Europeo y del Consejo (RGPD) y en la Ley Orgánica 3/2018 de Protección de Datos Personales y garantía de los derechos digitales (LOPDGDD), se informa de que los datos personales que puedan recogerse a través de este sitio web serán tratados por:

Responsable: NewForm Films
Proyecto / Marca: NewForm Films
Sitio web: www.newformfilms.com
Ubicación: Madrid, España
Contacto: https://newformfilms.com/contact

2. Finalidad del tratamiento de los datos

Los datos personales que el usuario facilite a través de este sitio web podrán ser utilizados para las siguientes finalidades:

  • Atender consultas o solicitudes enviadas a través de formularios de contacto o correo electrónico.
  • Mantener comunicación con usuarios interesados en el proyecto o servicios ofrecidos.
  • Mejorar la experiencia de navegación y analizar el uso del sitio web.

No se tomarán decisiones automatizadas basadas en los datos proporcionados por los usuarios.

3. Datos que se pueden recopilar

Dependiendo de la interacción del usuario con el sitio web, se podrán recopilar las siguientes categorías de datos:

  • Nombre
  • Dirección de correo electrónico
  • Información incluida en los mensajes enviados por el usuario
  • Datos técnicos de navegación (dirección IP, navegador, dispositivo, etc.)

4. Base legal para el tratamiento

El tratamiento de los datos personales se basa en:

  • El consentimiento del usuario, al enviar voluntariamente sus datos a través de formularios o correo electrónico.
  • El interés legítimo del responsable para mejorar el funcionamiento y seguridad del sitio web.

5. Conservación de los datos

Los datos personales se conservarán únicamente durante el tiempo necesario para cumplir con la finalidad para la que fueron recopilados o mientras exista una relación o comunicación activa con el usuario.

6. Destinatarios de los datos

Los datos personales no serán cedidos a terceros salvo obligación legal.

No obstante, el sitio web puede utilizar servicios técnicos de terceros necesarios para su funcionamiento, como:

  • servicios de alojamiento web
  • herramientas de analítica web
  • proveedores de correo electrónico

Estos proveedores podrán tener acceso a los datos únicamente para prestar sus servicios y bajo las correspondientes garantías legales.

7. Derechos de los usuarios

El usuario tiene derecho a:

  • Acceder a sus datos personales
  • Solicitar la rectificación de datos inexactos
  • Solicitar la supresión de sus datos cuando ya no sean necesarios
  • Solicitar la limitación del tratamiento
  • Oponerse al tratamiento de sus datos
  • Solicitar la portabilidad de los datos

Para ejercer estos derechos, el usuario puede enviar una solicitud al correo electrónico indicado anteriormente.

Asimismo, si considera que sus derechos no han sido respetados, puede presentar una reclamación ante la Agencia Española de Protección de Datos (www.aepd.es).

8. Seguridad de los datos

El responsable adopta las medidas técnicas y organizativas necesarias para garantizar la seguridad de los datos personales y evitar su alteración, pérdida, tratamiento o acceso no autorizado.

9. Cambios en la política de privacidad

El responsable del sitio web se reserva el derecho de modificar la presente Política de Privacidad para adaptarla a novedades legislativas o cambios en el funcionamiento del sitio web.

Se recomienda revisar periódicamente esta página para conocer cualquier actualización.
